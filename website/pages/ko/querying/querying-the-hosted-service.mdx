서브그래프가 배포된 상태에서 [호스팅 서비스](https://thegraph.com/hosted-service/)를 방문하여 쿼리를 실행하고 스키마를 확인하여 서브그래프에 대해 배포된 GraphQL API를 탐색할 수 있는[GraphiQL](https://github.com/graphql/graphiql) 인터페이스를 엽니다.

아래에 예가 제공되지만 서브그래프의 엔터티를 쿼리하는 방법에 대한 전반적인 내용은 [Query API](/querying/graphql-api)를 참조하시기 바랍니다.

#### 예시

이 쿼리는 매핑이 생성한 모든 카운터를 나열합니다. 우리가 하나만 생성하기 때문에 결과에는 `default-counter`가 하나만 포함됩니다.

```graphql
{
  counters {
    id
    value
  }
}
```

## 호스팅 서비스 사용하기

The Graph 탐색기 및 GraphQL 플레이그라운드는 호스팅 서비스에 배포된 서브그래프를 탐색하고 쿼리하는 데 유용한 방법입니다.
