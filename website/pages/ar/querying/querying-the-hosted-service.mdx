---
title: Querying the Hosted Service
---

مع نشر الـ subgraph ، قم بزيارة [Hosted Service](https://thegraph.com/hosted-service/) لفتح واجهة [GraphiQL](https://github.com/graphql/graphiql) حيث يمكنك استكشاف GraphQL API المنشور لـ subgraph عن طريق إصدار الاستعلامات وعرض المخطط.

An example is provided below, but please see the [Query API](/querying/graphql-api) for a complete reference on how to query the subgraph's entities.

#### مثال

يسرد هذا الاستعلام جميع العدادات التي أنشأها الـ mapping الخاص بنا. نظرا لأننا أنشأنا واحدا فقط ، فستحتوي النتيجة فقط على `default-counter`:

```graphql
{
  counters {
    id
    value
  }
}
```
