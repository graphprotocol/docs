---
title: Interroger le Service Hébergé
---

Une fois le subgraphe déployé, visitez le [Service Hébergé](https://thegraph.com/hosted-service/) pour ouvrir une interface [GraphiQL](https://github.com/graphql/graphiql) où vous pouvez explorer l'API GraphQL déployée pour le subgraphe en émettant des requêtes et en visualisant le schema.

Un exemple est fourni ci-dessous, mais veuillez consulter l'[API de requête](/querying/graphql-api) pour une référence complète sur la façon d'interroger les entités du subgraphe.

#### Exemple

Cette requête liste tous les compteurs que notre mapping a créés. Comme nous n'en créons qu'un seul, le résultat ne contiendra que notre seul `compteur par défaut` :

```graphql
{
  counters {
    id
    value
  }
}
```
