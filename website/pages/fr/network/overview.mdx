Le réseau The Graph est un protocole d'indexation décentralisé qui a pour but d'organiser les données de la blockchain. Les applications utilisent alors GraphQL pour interroger des API ouvertes appelées subgraphes, afin de récupérer les données indexées sur le réseau. Par conséquent, les développeurs peuvent créer des applications qui ne nécessitent pas de serveur et qui fonctionnent entièrement sur l'infrastructure publique.

## Vue d'ensemble

Le réseau The Graph se compose d'indexeurs, de curateurs et de délegateurs qui fournissent des services au réseau et servent des données aux applications Web3. Les consommateurs, quant à eux, utilisent les applications et consomment les données.

![Token Economics](/img/Network-roles@2x.png)

To ensure economic security of The Graph Network and the integrity of data being queried, participants stake and use Graph Tokens ([GRT](/tokenomics)). GRT is a work utility token that is an ERC-20 used to allocate resources in the network.

Active Indexers, Curators and Delegators can provide services and earn income from the network, proportional to the amount of work they perform and their GRT stake.
