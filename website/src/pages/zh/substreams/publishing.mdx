学习如何发布子流包到[子流注册](https://substreams.dev)。

## 概述

### 什么是包？

子流包是一个预编译的二进制文件，定义你想要从区块链中提取的特定数据，类似于传统子图中的`mapping.ts`文件。

## 发布包

### 先决条件

- 您必须安装子流 CLI 。
- 您必须有一个您想要发布的子流包(`.spkg`)。

### 第 1 步：运行 `substreams publish` 命令

1. 在命令行终端中，运行 `substreams publish <YOUR-PACKAGE>.spkg`。

2. 如果您的电脑没有设置代币，请导航到 `https://substreams.dev/me` 。

![get token](/img/1_get-token.png)

### 第 2 步：获取子流注册表中的代币

1. 在子流注册表中，使用您的 GitHub 帐户登录。

2. 创建一个新的代币并在安全的位置复制它。

![new token](/img/2_new_token.png)

### 第 3 步：在子流 CLI 中进行身份验证

1. 回到子流 CLI 中，粘贴之前生成的代币。

![paste token](/img/3_paste_token.png)

2. 最后，确认您想要发布这个包。

![confirm](/img/4_confirm.png)

就是这样！您已经成功地在子流注册表中发布了一个包。

![success](/img/5_success.png)
