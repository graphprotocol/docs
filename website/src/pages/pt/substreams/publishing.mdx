Aprenda a editar um pacote do Substreams no [Registo do Substreams](https://substreams.dev).

## Visão geral

### O que é um pacote?

Um pacote Substreams é um arquivo binário pré-compilado que define os dados específicos que você deseja extrair da blockchain, similar ao arquivo `mapping.ts` em subgraphs tradicionais.

## Edite um Pacote

### Pré-requisitos

- CLI do Substreams instalada
- Pacote do Substreams (`.spkg`) que deseja editar

### Etapa 1: Execute o comando `substreams publish`

1. Num terminal de linha de comando, execute `substreams publish <YOUR-PACKAGE>.spkg`.

2. Se não tiver um token definido no seu computador, navegue para `https://substreams.dev/me`.

![get token](/img/1_get-token.png)

### Passo 2: Adicione um Token ao Registo do Substreams

1. No Registo do Substreams, entre com a sua conta do GitHub.

2. Crie um novo token e copie-o num local seguro.

![novo token](/img/2_new_token.png)

### Passo 3: Autentique na CLI do Substreams

1. Na CLI do Substreams (Substreams), cole o token gerado previamente.

![colar token](/img/3_paste_token.png)

2. Por último, confirme que deseja editar o pacote.

![confirmar](/img/4_confirm.png)

Parabéns! Está editado com êxito um pacote no registo do Substreams.

![êxito](/img/5_success.png)
