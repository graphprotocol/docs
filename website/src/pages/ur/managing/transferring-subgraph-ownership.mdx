title: سب گراف کی ملکیت کی منتقلی
---

گراف سب گراف کی ملکیت کی منتقلی پرحمایت کرتا ہے.

جب آپ مینیٹ پر سب گراف تعینات کرتے ہیں، ایک NFT اس پتے پر لگایا جائے گا جس نے سب گراف کو تعینات کیا ہے۔ NFT ایک معیاری ERC721 پر مبنی ہے، تاکہ اسے آسانی سے مختلف کھاتوں میں منتقل کیا جا سکے.

جو بھی NFT کا مالک ہے سب گراف کو کنٹرول کرتا ہے۔ اگر مالک NFT فروخت کرنے یا اسے منتقل کرنے کا فیصلہ کرتا ہے، تو وہ نیٹ ورک پر اس سب گراف میں مزید ترمیم یا اپ ڈیٹ نہیں کر سکیں گے.

ڈیولپمنٹ لائف سائیکل میں مزید لچک پیدا کرنے کے علاوہ، یہ فعالیت بعض استعمال کے معاملات کو زیادہ آسان بناتی ہے، جیسے کہ آپ کے کنٹرول کو ملٹی سیگ میں منتقل کرنا یا DAO کی جانب سے کمیونٹی ممبر بنانا.

## اپنے سب گراف کو بطور NFT دیکھنا

اپنے سب گراف کو بطور NFT دیکھنے کے لیے، آپ NFT مارکیٹ پلیس جیسے OpenSea پر جا سکتے ہیں:

```
https://opensea.io/your-wallet-address
```

یا والٹ ایکسپلورر جیسا کہ **Rainbow.me**:

```
https://rainbow.me/your-wallet-addres
```
