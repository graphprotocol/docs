[Graph Assistant](https://thegraph.com/graph-assistant) (ベータ版)は、コード不要でブロックチェーンデータに即座にアクセスできるツールです。自然言語を、AIを活用した正確なクエリに変換します。

## 概要

[Graph Assistant](https://thegraph.com/graph-assistant)は、自然言語を使ってブロックチェーンデータを照会するための、高度で使いやすい [エージェント](/ja/ai-suite/graph-assistant/faq/)-ベースのチャットインターフェイスです。一般的なチャットボットとは異なり、ユーザーの意図を理解し、複雑なクエリを構築して自動的に実行することができます。これにより、コードを書くことなく、オンチェーンデータから豊富なAIを活用したインサイトを得ることができます。

- ユーザーの意図を**自動的に**解釈し、正確なクエリを生成して、The Graph の Subgraph や Token API 経由で実行します。そして、コードを一切必要とせずに、AIを活用した構造化された回答を提示します。
- 従来のダッシュボードやチャットボットに見られる技術的な複雑さを**取り除き**、インデックス化された Subgraph やトークンデータセットから、構成可能なインサイトを直接提供します。

## 主要機能

| 機能 | 説明 |
| --- | --- |
| **使いやすいインターフェイス** | 自然言語を介してブロックチェーンデータに会話形式でアクセスできます。質問するだけで、Assistant が回答を取得し、結果を表示します。 |
| **Subgraph によるサポート** | トランザクション、イベント、プロトコルのメトリクスなど、プロトコル固有のデータをインデックス化するために Subgraph を使用します。 |
| **Token API によるサポート** | 残高、送金、メタデータなど、トークンレベルの情報を得るために Token API を使用します。 |
| **セットアップ不要** | データソースの設定やクエリの作成は必要ありません。 |
| **自動クエリ構築** | 過去のトークン量、プロトコルの分析、クロスチェーンデータなどの複雑なクエリを実行し、フィルタリング、ページネーション、および Subgraph 全体にわたる結合を自動的に処理します。 |
