## 概要

チャットインターフェイスに、以下のような自然言語で質問を入力します。

```yaml
Uniswap で、TVL [Total Value Locked] が最も高いプールを USD 建てでトップ10表示して
```

> [!IMPORTANT] すべてのユーザーは、1時間あたり10件まで無料でクエリを実行できます。

> [!NOTE] 免責事項：Graph Assistant を利用することで、お客様のリクエストがログに記録され、基礎となるエージェントのパフォーマンス向上に使用される可能性があることに同意したことになります。機密情報、個人情報、プライベートな情報を共有しないでください。

### 仕組み

Assistant は以下の処理を行います。

1. ユーザーの意図を解釈する
2. 正しい Subgraph または Token API エンドポイントにマッピングする
3. 最適化されたクエリを実行する
4. きれいで構造化された結果を返す

クエリを作成したり、Subgraph のスキーマを理解したりする必要はありません。ただ質問するだけです。

### 使用例

**プロトコルの分析**

- 「2025年5月の Aave と Compound の日間アクティブユーザーを比較して」
- 「過去1四半期の XYZDAO の TVL [Total Value Locked] を表示して」

**トークンのインサイト**

- 「現在 Ethereum で時価総額がトップ10のトークンは？」
- 「5月1日から5月15日までの間に受け取った XYZ の数は？」

**過去データの取得**

- 「過去6か月の XYZ のトークン価格履歴を取得して」
- 「4月中の Ethereum の平均ガス価格はいくらだった？」

**ポートフォリオの監視**

- 「私のウォレットの現在のトークン残高と、それに対応する USD 相当額を一覧表示して」
- 「XYZ の特定の日付以降の損益を計算して」

**クロスチェーンの比較**

- 「Ethereum と Polygon での XYZ の7日間の取引量を表示して」
- 「昨日 Arbitrum で行われた XYZ のスワップ数は？」

利用を開始するには、[クイックスタート](https://www.google.com/search?q=/ai-suite/graph-assistant/quick-start/)の手順に従い、その機能を試してみてください。
