> [!IMPORTANT] Graph Assistant はまだ **ベータ版** です。破壊的変更、レート制限、および一時的なダウンタイムが発生する可能性があることをご了承ください。

このガイドでは、Graph Assistant を素早く使い始めるための手順を説明します。

## ステップバイステップ

### ステップ1. アカウントを作成する

以下のサイトにアクセスします：[Graph Assistant](https://thegraph.com/graph-assistant)

- 「Sign Up」（または、アカウントをお持ちの場合は「Log In」）をクリックします。
- Google、GitHub、X、メール、またはウォレットの認証情報を使ってログインします。
- 追加のセットアップは必要ありません。

### ステップ2. Assistant のチャット画面を開く

ログイン後:

- Graph Assistant のインターフェイスが表示されます。
- ページ下部にチャット入力バーがあります。ここで Assistant とやり取りを行います。

### ステップ3：最初の質問をする

> [\!IMPORTANT] すべてのユーザーは、1時間あたり10件まで無料でクエリを実行できます。

入力バーに、自然言語（日常の言葉）でクエリを入力します。

例:

```yaml
Ethereum 上の XYZ トークンの24時間取引量を表示して
```

> [!NOTE] 免責事項：Graph Assistant を利用することで、お客様のリクエストがログに記録され、AIインターフェイスのパフォーマンス向上に使用される可能性があることに同意したことになります。機密情報、個人情報、プライベートな情報を共有しないでください。

Assistant は以下の処理を行います。

- 質問を理解する
- Subgraph または Token API を使ってオンチェーンデータを取得する
- 必要なデータを含む、きれいでフォーマットされた回答を返す
