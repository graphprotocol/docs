## Token Endpoints - $15/mil queries

Query token transfers, balances, and metadata across EVM and SVM blockchains

```
GET /v1/evm/tokens
GET /v1/svm/tokens
GET /v1/evm/transfers
GET /v1/svm/transfers
GET /v1/evm/balances
GET /v1/svm/balances
GET /v1/evm/balances/native
GET /v1/svm/balances/native
```

## DEX Endpoints - $50/mil queries

Access on-chain swap, liquidity pool, and decentralized exchange data

```
GET /v1/evm/swaps
GET /v1/svm/swaps
GET /v1/evm/pools
GET /v1/svm/pools
```
