서브그래프는 그래프가 이더리움에서 인덱싱할 데이터와 데이터를 저장하는 방법을 정의합니다. 일단 배포되면, 이는 블록체인 데이터의 글로벌 그래프의 일부를 형성합니다.

![서브그래프 정의](/img/defining-a-subgraph.png)

서브그래프 정의는 몇 가지 파일로 구성됩니다:

- `subgraph.yaml`: 서브그래프 매니페스트를 포함하는 YAML 파일

- `schema.graphql`: 서브그래프에 저장되는 데이터와 GraphQL을 통해 쿼리하는 방법을 정의하는 GraphQL 스키마

- `AssemblyScript Mappings`: [AssemblyScript](https://github.com/AssemblyScript/assemblyscript) 이벤트 데이터를 스키마에 정의된 엔터티로 변환하는 코드(e.g. `mapping.ts` in this tutorial)

매니페스트 파일의 내용에 대해 자세히 알아보기 전에 [Graph CLI](https://github.com/graphprotocol/graph-cli)를 설치해야 합니다. 이는 서브그래프를 구축하고 배포하는데 필요합니다.

## Graph CLI 설치

Graph CLI는 JavaScript로 작성되었으며 이를 사용하려면 `yarn` 또는 `npm`을 설치해야 합니다; 아래와 같이 yarn 이 있다고 가정합니다.

`yarn`이 있으신 경우, Graph CLI를 실행하여 설치합니다.

**yarn과 함께 설치합니다:**

```bash
yarn global add @graphprotocol/graph-cli
```

**npm과 함께 설치합니다:**

```bash
npm install -g @graphprotocol/graph-cli
```
