---
title: Query the Hosted Service
---

With the subgraph deployed, visit the [Hosted Service](https://thegraph.com/hosted-service/) to open up a [GraphiQL](https://github.com/graphql/graphiql) interface where you can explore the deployed GraphQL API for the subgraph by issuing queries and viewing the schema.

Một ví dụ được cung cấp bên dưới, nhưng vui lòng xem [Query API](/querying/graphql-api) để có tài liệu tham khảo đầy đủ về cách truy vấn các thực thể của subgraph.

#### Ví dụ

Truy vấn này liệt kê tất cả các bộ đếm mà ánh xạ của chúng tôi đã tạo. Vì chúng tôi chỉ tạo một, kết quả sẽ chỉ chứa một `default-counter`:

```graphql
{
  counters {
    id
    value
  }
}
```
